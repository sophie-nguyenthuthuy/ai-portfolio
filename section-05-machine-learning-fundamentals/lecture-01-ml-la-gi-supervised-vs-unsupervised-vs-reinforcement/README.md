### Slide 2 — Lập trình truyền thống vs ML

- Truyền thống: dữ liệu + quy tắc → kết quả
- ML: dữ liệu + kết quả → quy tắc (model)

**🎨 Visual:** `[Mermaid]` So sánh 2 luồng.
**🎤 Speaker note:** Đây là thay đổi tư duy cốt lõi.
