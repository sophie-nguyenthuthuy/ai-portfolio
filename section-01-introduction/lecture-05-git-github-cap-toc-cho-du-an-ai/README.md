- Git = công cụ quản lý phiên bản (máy bạn)
- GitHub = nơi lưu & chia sẻ online

**🎨 Visual:** `[Mermaid]` Máy local ↔ GitHub cloud.
**🎤 Speaker note:** Phân biệt rõ 2 khái niệm hay nhầm.

**🎨 Visual:** `[AI image]` Trang profile GitHub đẹp.

**🎤 Speaker note:** 90% công việc hàng ngày chỉ cần 4 lệnh này.
