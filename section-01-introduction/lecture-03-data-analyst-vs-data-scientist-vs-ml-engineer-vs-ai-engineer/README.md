# Section 1 · Lecture 3 — Data Analyst vs Data Scientist vs ML Engineer vs AI Engineer

**Số slide:** 7 · ~5 phút

---

## Nội dung slide

### Slide 1 — Tiêu đề

- 4 vai trò — khác nhau thế nào?

**🎨 Visual:** `[AI image]` 4 nhân vật làm 4 việc khác nhau.
**🎤 Speaker note:** "Nhiều người nhầm lẫn — hôm nay mình phân biệt rõ."

### Slide 2 — Data Analyst

- Trả lời "chuyện gì đã xảy ra?"
- Công cụ: SQL, Excel, Power BI
- Output: dashboard, báo cáo

**🎨 Visual:** `[Mermaid]` Dữ liệu → dashboard.
**🎤 Speaker note:** Điểm vào dễ nhất, đặc biệt cho người non-tech.

### Slide 3 — Data Scientist

- Trả lời "tại sao & dự đoán gì?"
- Công cụ: Python, ML, thống kê
- Output: model dự đoán, insight

**🎨 Visual:** `[Mermaid]` Dữ liệu → model → dự đoán.
**🎤 Speaker note:** Nặng phân tích + mô hình hoá.

### Slide 4 — ML Engineer

- Đưa model vào sản phẩm
- Công cụ: Python, API, MLOps
- Output: hệ thống ML chạy production

**🎨 Visual:** `[Mermaid]` Model → API → production.
**🎤 Speaker note:** Cầu nối giữa data science và software engineering.

### Slide 5 — AI Engineer

- Xây ứng dụng trên LLM/GenAI
- Công cụ: RAG, agent, fine-tuning
- Output: chatbot, agent, sản phẩm AI

**🎨 Visual:** `[Mermaid]` LLM + RAG + tools → ứng dụng AI.
**🎤 Speaker note:** Vai trò hot nhất 2026 — khoá này nhắm tới đây.

### Slide 6 — Bảng so sánh nhanh

- 4 cột: câu hỏi · công cụ · output · mức lương

**🎨 Visual:** `[Mermaid table]` Bảng so sánh 4 vai trò.
**🎤 Speaker note:** Chụp lại slide này để tham khảo.

### Slide 7 — Chọn hướng & chuyển bài

- Không cần chọn ngay — khoá phủ cả 4
- Bài tiếp: cài đặt môi trường →

**🎨 Visual:** `[AI image]` La bàn / ngã rẽ.
**🎤 Speaker note:** "Học hết, bạn sẽ tự biết mình hợp vai trò nào."
